# Keeper App – Full Stack Project

A full-stack task manager where users can sign up, log in, and manage tasks using a clean React UI and secure Node.js backend.

---

## Features
- User Signup & Login (JWT Auth)
- Create, update, delete tasks
- Drag & drop between TODO / DOING / DONE
- React UI with modern CSS styling
- Node.js + Express backend API
- PostgreSQL database with structured schema
- Fully responsive design
